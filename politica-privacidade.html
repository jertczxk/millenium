2. DADOS COLETADOS

2.1 Dados Pessoais

  • Nome completo
  • CPF
  • Data de nascimento
  • E-mail
  • Telefone/WhatsApp
  • Endereço residencial
  • Dados de dependentes (quando aplicável)

2.2 Dados Sensíveis (Saúde)

Em conformidade com a LGPD (Art. 11), coletamos dados de saúde APENAS para prestação dos serviços de telemedicina, mediante consentimento expresso e específico.

Tipos de dados sensíveis:

  • Histórico médico relatado em consultas
  • Prescrições médicas
  • Registros de atendimento teleconsulta

2.3 Dados de Navegação

  • Cookies (veja nossa Política de Cookies)
  • IP, geolocalização aproximada
  • Dispositivo e navegador utilizados

3. FINALIDADE DO TRATAMENTO

Utilizamos seus dados para:

  • ✓ Prestação dos serviços contratados
  • ✓ Emissão de boletos e gestão financeira
  • ✓ Comunicação sobre planos e benefícios
  • ✓ Acionamento de auxílio funeral (emergencial)
  • ✓ Melhorias na experiência do usuário
  • ✓ Cumprimento de obrigações legais

4. COMPARTILHAMENTO DE DADOS

Seus dados podem ser compartilhados com:

  • Clínicas/Médicos parceiros: Para agendamento de consultas
  • Rede de benefícios: Para aplicação de descontos
  • Funerárias parceiras: APENAS em caso de acionamento
  • Processadores de pagamento: Para cobrança
  • Autoridades: Quando exigido por lei

NÃO vendemos, alugamos ou comercializamos seus dados pessoais.

5. SEUS DIREITOS (LGPD)

Você pode solicitar:

  • ✓ Confirmação de tratamento de dados
  • ✓ Acesso aos seus dados
  • ✓ Correção de dados incompletos/incorretos
  • ✓ Anonimização, bloqueio ou eliminação
  • ✓ Portabilidade dos dados
  • ✓ Revogação do consentimento

Como exercer: privacidade@milleniumprev.com.br

6. SEGURANÇA

Adotamos medidas técnicas e administrativas:

  • Criptografia SSL/TLS em todas as transmissões
  • Servidores em ambiente seguro (certificações ISO)
  • Acesso restrito por credenciais
  • Backups regulares
  • Treinamento da equipe em proteção de dados

7. RETENÇÃO DE DADOS

  • Dados cadastrais: Enquanto o contrato estiver ativo + 5 anos (prescrição legal)
  • Dados de saúde: 20 anos (CFM 1.821/2007)
  • Dados financeiros: 5 anos (legislação tributária)

8. COOKIES E TECNOLOGIAS

Veja detalhes na nossa Política de Cookies.
